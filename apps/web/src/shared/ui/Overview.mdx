# sy-universe UI Atoms

`shared/ui`에 있는 공통 컴포넌트 목록입니다. 왼쪽 사이드바 **Atoms** 아래에서 확인할 수 있어요.

| Atom | 용도 |
|------|------|
| Button | 액션 버튼 (ghost / accent / outline) |
| Input / Textarea / Label | 폼 입력 |
| Badge | 기간·상태 메타 |
| Chip / ChipRow | 스택·태그 |
| Eyebrow | 패널 상단 카테고리 라벨 |
| Modal | 다이얼로그 |
| Panel | 상세 패널(스토리 생략 — 페이지 전용) |
| IconButton | 아이콘용 정사각 버튼 |
| Spinner | 로딩 |
| SpaceBackdrop | 우주 배경(페이지 전용) |

## 콘텐츠(노드)는 여기가 아님

회사/프로젝트 추가는 Storybook이 아니라:

- 회사 → `public/content/portfolio.json`
- 프로젝트 → `public/content/projects/*.json` + `manifest.json`

자세한 트리는 루트 `README.md`를 보세요.
